# ReplaceImage
ReplaceImage is a chrome extension that allows you to replace images of a page with any term you like using the openverse api. You can use it as a prank on friends/family/coworkers or to just have fun with it yourself. 

## Installation
1. Clone the repository to your local machine.
2. Open Google Chrome and navigate to `chrome://extensions/`.
3. Enable "Developer mode" by toggling the switch in the top right corner.
4. Click on "Load unpacked" and select the directory where you cloned the repository.
5. The extension should now be installed and ready to use.
